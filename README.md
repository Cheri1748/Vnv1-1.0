
## 可视导航场面目标检测器
## Vn_v1.0.0

## 1.简介

可视导航项目中场面威胁目标检测的官方实现。可检测场面目标，包括飞机、特种车辆和机务工作人员。并对目标测距。检测模块使用yolov5，追踪模块使用deepsort，测距采用基于相机位姿的测距原理。
